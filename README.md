# 💖 MAHAL – Memory & Events Website

MAHAL is a personal memory and event compilation website designed to showcase special moments, photos, and experiences from different occasions such as café dates, travels, and celebrations.

It works like a digital scrapbook where every event has its own dedicated gallery page.

---

## ✨ Features

- 🏠 Modern homepage with hero section
- 📸 Memory gallery with clickable images
- 🗂️ Event-based organization (each event has its own page)
- ☕ Dedicated café memory page (Estratto event)
- 🖼️ Clickable image gallery with modal preview
- 📱 Mobile-friendly swipeable image viewer
- 🎨 Custom styled UI with dark romantic theme
- 💖 Floating heartbeat animated button
- 🔗 Easy navigation between pages
